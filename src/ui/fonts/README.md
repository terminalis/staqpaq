# Vendored Fonts

These WOFF2 files are local copies of the Google Fonts families previously loaded at runtime:

- VT323 by Peter Hull
- Share Tech Mono by Carrois Apostrophe

Both families are distributed under the SIL Open Font License. Source metadata and current
upstream releases are available from Google Fonts:

- https://fonts.google.com/specimen/VT323
- https://fonts.google.com/specimen/Share+Tech+Mono
