# Vendored Icons

`icon-data.js` contains only the Iconify SVG records used by staqpaq source templates and
`data/catalogue.json`. It is intentionally selected-only, not a full icon collection dump.

Current source collections:

- Simple Icons (`simple-icons:*`), CC0-1.0, primary brand/provider source.
- Solar (`solar:*`), CC-BY-4.0, UI/action icons.
- Material Design Icons (`mdi:*`), Apache-2.0, generic non-brand fallback icons.
- SVG Logos (`logos:*`), CC0-1.0, brand fallbacks when Simple Icons has no current match.
- Huge Icons (`hugeicons:*`), MIT, one provider fallback.

Source metadata is available from Iconify collection metadata:

- https://icon-sets.iconify.design/simple-icons/
- https://icon-sets.iconify.design/solar/
- https://icon-sets.iconify.design/mdi/
- https://icon-sets.iconify.design/logos/
- https://icon-sets.iconify.design/hugeicons/
